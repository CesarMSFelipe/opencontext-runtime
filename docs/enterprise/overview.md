# Overview

## Purpose
OpenContext is enterprise-ready in design, not yet a complete enterprise runtime. It has safe defaults, policy models, local traces, evidence scaffolds, release audits, prompt audits, run receipts, and deny-by-default tool/provider boundaries.

## Current Status
Design and local scaffolds are present. Do not treat the project as a fully certified enterprise platform yet.

## Related Commands
```bash
opencontext report
opencontext release evidence
opencontext org baseline check
opencontext release gate
```
